# ArchiveOS Control Tower Wireframe / 화면설계도

This is a Wireframe / 화면설계도, not an actual screenshot.

## Layout

- Top KPI: ecosystem status, pending approvals, callback pending, callback failed
- Service status cards: Nexus, Archive-Logistics, Archive-Ledger, ArchiveOS
- Topology panel: Nexus -> Logistics -> Ledger -> ArchiveOS callback
- Ledger Approval Queue: approval request list and selected detail
- Callback Outbox: pending, retry, failed, sent callbacks
- Policy Evidence Panel: RAG evidence or fallback evidence
- Timeline: ecosystem events and audit-linked history
- Safe-mode banner: external write protection state
