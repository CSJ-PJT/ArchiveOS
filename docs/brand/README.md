# ArchiveOS Brand

ArchiveOS uses a geometric `A` mark with a cyan decision core.

## Assets

- `archiveos-mark.svg` — square mark for app icon, favicon, sidebar, and compact UI surfaces.
- `archiveos-lockup.svg` — horizontal README and product lockup.

## Usage

```html
<img src="docs/brand/archiveos-lockup.svg" width="720" alt="ArchiveOS" />
```

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Background | `#071014` | dark console background |
| Surface | `#0F1214` | cards and elevated surfaces |
| Text | `#F5F7FA` | mark and primary text |
| Cyan Core | `#16D6E3` | decision core and active accents |
| Muted Border | `#1B2126` | panels and dividers |
