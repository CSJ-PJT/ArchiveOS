# Ecosystem Diagrams

This directory contains Mermaid source files and rendered images for the Archive Platform ecosystem.

All diagrams use synthetic/demo data concepts only. External service names use `Archive-Logistics`. Internal values such as `LOGITICS`, `logitics`, and `ARCHIVE_ECOSYSTEM_SERVICES_LOGITICS_*` are mentioned only as compatibility keys for existing contracts.

## Files

- `01-ecosystem-architecture`: full ecosystem architecture.
- `02` to `04`: end-to-end and approval sequences.
- `05` to `08`: service component diagrams.
- `09` to `11`: ERD diagrams.
- `12` to `14`: state transitions.
- `15`: local compose runtime topology.
- `16` to `18`: before/after troubleshooting diagrams.

Rendered SVG/PNG files are generated with Mermaid CLI.
